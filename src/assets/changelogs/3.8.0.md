	- Combined the organisations page with the overview page for a more cohesive experience

- Added a new clear notifications button
- Added a query runtime to the Query view to show how long your whole request has been running

- Resolved non-functioning keybinds on macOS
- Resolved reported crashes in the Designer view
- Resolved Explorer view crashes caused by stale data

- Prevented the escape key from exiting full screen on macOS
- Improved syntax highlighting colors in light mode when searching
- Added missing horizontal and vertical scrollbars to the Explorer view
- Stopped record selections from persisting across tables
- Improved scrolling performance with large namespaces and databases
- Ensured decimal points below 0.000 are displayed correctly
- Fixed record IDs incorrectly rendering in light mode when app is in dark mode
- Made sure error messages are properly displayed in the record inspector
